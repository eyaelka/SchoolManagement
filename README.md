# SchoolManagement

## backend : Nodejs
To run the backend Project : open backend folder
Open terminal: 
$ npm install
$ npm start

	
Visit http://localhost:3000

## frontend : Angular
To run the frontend Project : open frontend folder
Open terminal: 
$ npm install
$ ng serve
	
Visit http://localhost:4200
